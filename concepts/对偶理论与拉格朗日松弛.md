# 对偶理论与拉格朗日松弛 (Lagrangian Duality & Relaxation)

> **一句话**：对偶理论将优化问题分为**原问题**（Primal）和**对偶问题**（Dual）——对偶问题给出原问题最优值的下界（或上界）。在安全控制中，拉格朗日松弛用于将**硬约束 QP 转化为软约束**，并构成 [[可微 QP (Differentiable QP)]] 和 [[KKT 条件]] 的理论基础。

---

## 1. 直觉理解

### 1.1 拍卖类比

想象一个拍卖：

- **原问题**：买家想以最低价买到满足所有要求的商品
- **对偶问题**：卖家的"影子价格"——如果放松某个要求，能省多少钱？

对偶间隙 = 买家的最低价 - 卖家的影子价格总和 ≥ 0

如果对偶间隙 = 0 → **强对偶性** → 两个问题的最优值相同。

### 1.2 图示

```
原问题最优值 p*
 ──────────────── ← 上界
    对偶间隙 ≥ 0
 ──────────────── ← 下界
对偶问题最优值 d*

p* ≥ d* (弱对偶性)
p* = d* (强对偶性, 凸问题通常成立)
```

## 2. 形式定义

### 2.1 原问题

$$\min_x \quad f(x)$$
$$\text{s.t.} \quad g_i(x) \leq 0, \quad i = 1, \ldots, m$$
$$\quad\quad h_j(x) = 0, \quad j = 1, \ldots, p$$

### 2.2 Lagrangian

$$\mathcal{L}(x, \lambda, \nu) = f(x) + \sum_{i=1}^{m} \lambda_i g_i(x) + \sum_{j=1}^{p} \nu_j h_j(x)$$

### 2.3 对偶函数

$$g(\lambda, \nu) = \min_x \mathcal{L}(x, \lambda, \nu)$$

对偶函数给出原问题最优值的**下界**：

$$g(\lambda, \nu) \leq p^* \quad \text{对所有 } \lambda \geq 0$$

### 2.4 对偶问题

$$\max_{\lambda, \nu} \quad g(\lambda, \nu)$$
$$\text{s.t.} \quad \lambda \geq 0$$

---

## 3. 弱对偶与强对偶

### 3.1 弱对偶性

$$d^* \leq p^*$$

**总是成立**——对偶问题最优值 ≤ 原问题最优值。

### 3.2 强对偶性

$$d^* = p^*$$

**不总是成立**，但对于**凸问题**，在 Slater 条件（存在严格可行点）下成立。

### 3.3 对偶间隙

$$\text{Gap} = p^* - d^* \geq 0$$

- Gap = 0 → 强对偶
- Gap > 0 → 弱对偶（非凸问题可能出现）

## 4. 在安全控制中的应用

### 4.1 CBF-QP 的对偶

[[CBF (控制障碍函数)]] 的 QP：

$$\min_u \frac{1}{2}\|u - u_{\text{ref}}\|^2$$
$$\text{s.t.} \quad A u \leq b \quad (\text{CBF + 控制界约束})$$

**Lagrangian**：

$$\mathcal{L}(u, \lambda) = \frac{1}{2}\|u - u_{\text{ref}}\|^2 + \lambda^T (Au - b)$$

**对偶函数**：

$$g(\lambda) = \min_u \mathcal{L} = -\frac{1}{2}\|A^T \lambda\|^2 + (u_{\text{ref}}^T A^T - b^T)\lambda + \frac{1}{2}\|u_{\text{ref}}\|^2$$

**对偶问题**：

$$\max_\lambda g(\lambda) \quad \text{s.t.} \quad \lambda \geq 0$$

### 4.2 拉格朗日松弛 → 软约束

将硬约束 CBF-QP 转化为软约束：

$$\min_u \frac{1}{2}\|u - u_{\text{ref}}\|^2 + \rho \max(0, Au - b)^2$$

其中 $\rho$ 是惩罚系数。

这正是 [[BarrierNet]] 和 [[dCBF (可微控制障碍函数)]] 的核心思想！

### 4.3 增广拉格朗日法

$$\min_u \frac{1}{2}\|u - u_{\text{ref}}\|^2 + \lambda^T \max(0, Au - b) + \frac{\rho}{2} \|\max(0, Au - b)\|^2$$

交替更新 $u$ 和 $\lambda$：

```python
def augmented_lagrangian_qp(u_ref, A, b, rho=10.0, n_iter=50):
    """
    增广拉格朗日法求解 CBF-QP
    
    min 0.5||u - u_ref||^2
    s.t. Au <= b
    
    转化为:
    min 0.5||u - u_ref||^2 + λ' max(0, Au-b) + ρ/2 ||max(0, Au-b)||^2
    """
    import torch
    
    n = u_ref.shape[0]
    m = A.shape[0]
    
    u = u_ref.clone()
    lam = torch.zeros(m)
    
    for iteration in range(n_iter):
        # 更新 u (梯度下降)
        for inner in range(10):
            violation = torch.relu(A @ u - b)
            
            grad_u = (u - u_ref) + A.T @ (lam * (violation > 0).float()) + \
                     rho * A.T @ violation
            u = u - 0.01 * grad_u
        
        # 更新 λ (对偶上升)
        violation = torch.relu(A @ u - b)
        lam = lam + rho * violation
        lam = torch.relu(lam)  # λ >= 0
    
    return u

# 测试
u_ref = torch.tensor([-2.0])
A = torch.tensor([[-1.0], [1.0]])  # -u <= 0, u <= 3
b = torch.tensor([0.0, 3.0])

u_star = augmented_lagrangian_qp(u_ref, A, b)
print(f"u* = {u_star.item():.4f}")  # 应 ≈ -2.0 (如果可行)
```

## 5. 对偶变量（影子价格）的物理意义

### 5.1 在 CBF-QP 中

对偶变量 $\lambda_i$ 表示第 $i$ 个约束的**紧迫程度**：

- $\lambda_i = 0$：约束不活跃，控制器"不关心"
- $\lambda_i$ 大：约束很紧，控制器"被迫"满足

### 5.2 在 BarrierNet 中

BarrierNet 的惩罚系数 $p_i(z)$ 可以看作**可学习的对偶变量**：

- 传统 QP：$\lambda_i$ 由优化器求解
- BarrierNet：$p_i(z)$ 由神经网络预测

## 6. 代码：求解对偶问题

```python
import torch

def solve_dual_cbf_qp(H, F, G, h):
    """
    求解 CBF-QP 的对偶问题
    
    原问题: min 0.5 u'Hu + F'u s.t. Gu <= h
    对偶:   max -0.5 (G'λ + F)' H^{-1} (G'λ + F) - h'λ
            s.t. λ >= 0
    """
    n = H.shape[0]
    m = G.shape[0]
    
    H_inv = torch.linalg.inv(H)
    
    # 对偶目标函数
    def dual_objective(lam):
        w = G.T @ lam + F
        return -0.5 * w.T @ H_inv @ w - h.T @ lam
    
    # 梯度上升
    lam = torch.zeros(m, requires_grad=True)
    optimizer = torch.optim.Adam([lam], lr=0.01)
    
    for iteration in range(100):
        optimizer.zero_grad()
        obj = dual_objective(torch.relu(lam))  # λ >= 0
        (-obj).backward()
        optimizer.step()
    
    lam_star = torch.relu(lam.detach())
    
    # 从对偶解恢复原问题解
    u_star = -H_inv @ (G.T @ lam_star + F)
    
    return u_star, lam_star

# 测试
H = torch.tensor([[2.0]])
F = torch.tensor([-6.0])
G = torch.tensor([[-1.0], [1.0]])
h = torch.tensor([0.0, 3.0])

u_star, lam_star = solve_dual_cbf_qp(H, F, G, h)
print(f"u* = {u_star.item():.4f}")  # 3.0
print(f"λ* = {lam_star.tolist()}")
```

## 7. 对偶性总结

| 概念 | 原问题 | 对偶问题 |
|------|--------|---------|
| **目标** | min $f(x)$ | max $g(\lambda)$ |
| **变量** | $x$（原变量） | $\lambda$（对偶变量） |
| **约束** | $g(x) \leq 0$ | $\lambda \geq 0$ |
| **最优值** | $p^*$ | $d^* \leq p^*$ |
| **凸问题** | $p^* = d^*$ | 强对偶 |
